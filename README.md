# cxBattle
原型是上班摸鱼写在控制台里的小游戏，用AI重构了亿点点，绝大部分代码都是AI，包括readme与文档


一个基于 `Vue 3 + Vite + Electron` 的回合制战斗 Demo，支持：
- 难度系统（普通/困难/极难/专家/炼狱）
- 连战模式与连战敌人成长
- rougelike祝福与装备构筑系统（战前 6 选、战中三选一）
- Electron 本地 JSON 数据读写与自定义编辑

## 游戏性
- 战斗节奏：回合制对战，围绕“技能释放时机”和“生存/输出取舍”进行决策。
- 构筑选择：战前可进行多轮候选选择，提前规划祝福与装备方向，形成不同流派。
- 战中成长：战斗过程中会触发三选一强化，局内可持续调整构筑路线。
- 难度挑战：不同难度会显著影响敌方强度，适合从熟悉机制到极限挑战的渐进体验。
- 连战压力：击败敌人后进入下一战，敌人会成长，要求玩家在收益与风险间动态平衡。

## 快速开始

```bash
npm install
npm run dev
```

## 主要脚本

```bash
# Web 开发
npm run dev

# Electron 开发
npm run electron:dev

# 构建前端
npm run build
```
