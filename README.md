# cxBattle

一个基于 `Vue 3 + Vite + Electron` 的回合制战斗 Demo，支持：
- 难度系统（普通/困难/极难/专家/炼狱）
- 连战模式与连战敌人成长
- rougelike祝福与装备构筑系统（战前 6 选、战中三选一）
- Electron 本地 JSON 数据读写与自定义编辑

## 快速开始

```bash
npm install
npm run dev
```

## 主要脚本

```bash
# Web 开发
npm run dev

# Electron 开发
npm run electron:dev

# 构建前端
npm run build
```
